# NULLZONE

> Tetris meets 1010! inside a hacking narrative. Fill rows. Breach sectors. Don't let the corruption win.

**NULLZONE** is a block puzzle game with a cyberpunk soul. Place pieces on an 8×8 grid to clear rows and columns — but as you go deeper, the system fights back.

## What It Is

A browser-based puzzle game where players fill rows and columns to breach encrypted data sectors on a glitching grid. The deeper you go, the more the system resists: corruption tiles appear, rows lock, surges fire without warning. Six sector themes shift the visual palette from residential cyan to stark void as the tension builds.

This is a meditative tension loop — a casual puzzler that turns into something darker the longer you play.

## Features

- **8×8 grid puzzle** with Tetris and 1010! mechanics
- **Six sector themes** — each with a distinct visual palette and escalating hazards
- **Corruption, row locks, and surge mechanics** that evolve with depth
- **Leaderboard system** — compete across sessions
- **Daily Mode** — same puzzle seed for all players each day
- Single-file HTML — no install, no account, no data collection

## Genre & Tone

Puzzle / Cyberpunk / Arcade  
Meditative / Dark / Escalating Tension
